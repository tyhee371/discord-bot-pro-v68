# Discord Bot Pro - dependency list (Node.js project)
#
# NOTE:
# - This bot is written for Node.js (discord.js). A Python/pip "requirements.txt"
#   isn't used to install dependencies. Use npm instead.
#
# System requirements:
#   - Node.js >= 22.12.0
#   - npm (bundled with Node.js)
#
# Install:
#   1) In the bot folder, run:
#        npm install
#   2) Copy .env.example to .env and fill in your values
#   3) Deploy commands (recommended for testing):
#        npm run deploy:guild
#   4) Start:
#        npm start
#
# npm dependencies used by this bot (versions from package.json):
#   - discord.js 14.25.1
#   - keyv ^5.5.0
#   - @keyv/sqlite ^4.0.2
#
# One-liner install (optional):
#   npm install discord.js@14.25.1 keyv@^5.5.0 @keyv/sqlite@^4.0.2
