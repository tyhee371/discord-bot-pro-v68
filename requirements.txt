# Requirements and Installation Notes

This project is a Node.js Discord bot. Dependencies are managed via `package.json` and installed with `npm`.

## System Requirements

- Node.js 22.12.0 or higher
- npm (bundled with Node.js)
- FFmpeg for music playback
- Docker + Docker Compose for containerized deployment

## Install Dependencies

```bash
npm install --include=optional --legacy-peer-deps
```

## Optional Packages

Optional dependencies improve voice and music playback:

- `@discordjs/opus` — better voice performance than `opusscript`
- `yt-dlp` — improved YouTube playback fallback
- `ffmpeg-static` — included for cross-platform FFmpeg support

## Notes

- This file is informational only.
- Do not use `pip` for this repository.
- Use `npm run migrate` to run database migrations.
- Use `npm run deploy:guild` or `npm run deploy:global` to register Discord slash commands.
